# Трансформаторна підстанція 35/10 кВ «Лівобережна» 

Навчальний проєкт: інформаційний односторінковий вебсайт для енергетичного об'єкта (Варіант 11). 

## Опис проєкту
Цей проєкт розроблено в рамках практичної роботи зі створення вебсторінок. Сайт надає загальну інформацію про трансформаторну підстанцію «Лівобережна» (м. Дніпро), її призначення, технічні характеристики та контактні дані. 

## Основні функції та особливості
* **Адаптивний дизайн:** Сторінка коректно відображається на ПК, планшетах та мобільних пристроях завдяки сітці Bootstrap.
* **Зручна навігація:** Фіксоване навігаційне меню з інтегрованими якорями для швидкого переходу між розділами.
* **Структурований контент:** Детальний опис об'єкта, стилізована таблиця технічних параметрів та однолінійна електрична схема.
* **Сучасний UI/UX:** Використання градієнтів, hover-ефектів для таблиць і карток, а також CSS-анімацій (плавна поява елементів при завантаженні).

## Використані технології
* **HTML5:** Семантична розмітка сторінки.
* **CSS3:** Кастомні стилі, CSS-змінні, анімації (`@keyframes`).
* **Bootstrap 5.3.0:** Компоненти (Navbar, Cards, Tables) та адаптивна сітка.
